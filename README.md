# github-final-project
A calculator that calculates simple interest given principal, annual rate of interest and time period in years.
## Input:
<ul>
  <li>
   p, principal amount
  </li>
  <li>
   t, time period in years
  </li>
  <li>
   r, annual rate of interest
  </li>
  </ul>
<h2>Output</h2>
   simple interest = p*t*r
